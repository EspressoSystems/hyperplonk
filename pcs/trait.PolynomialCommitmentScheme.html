Expand description

This trait defines APIs for polynomial commitment schemes. Note that for our usage of PCS, we do not require the hiding property.

fn gen_srs_for_testing<R: RngCore>(
    rng: &mut R,
    log_size: usize
) -> Result<Self::SRS, PCSErrors>

Build SRS for testing.

  • For univariate polynomials, log_size is the log of maximum degree.
  • For multilinear polynomials, log_size is the number of variables.

WARNING: THIS FUNCTION IS FOR TESTING PURPOSE ONLY. THE OUTPUT SRS SHOULD NOT BE USED IN PRODUCTION.
