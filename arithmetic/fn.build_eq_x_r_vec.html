Function arithmetic::build_eq_x_r_vec

source · 
pub fn build_eq_x_r_vec<F: PrimeField>(r: &[F]) -> Result<Vec<F>, ArithErrors>
Expand description

This function build the eq(x, r) polynomial for any given r, and output the evaluation of eq(x, r) in its vector form.

Evaluate eq(x,y) = \prod_i=1^num_var (x_i * y_i + (1-x_i)(1-y_i)) over r, which is eq(x,y) = \prod_i=1^num_var (x_i * r_i + (1-x_i)(1-r_i))
